Was ist H5P?

AuthorNadine Rück
Date05. Jan 2023

H5P ist eine freie und quelloffene Software zum Erstellen von interaktiven (Lern-)Inhalten für das Web. Für die Erstellung von H5P-Inhalten gibt es einen Editor, so dass keine Programmierkenntnisse nötig sind. Die Inhalte können auf Webseiten, in Content-Management-Systemen (Wordpress und Drupal) und in Learning Management Systemen (LMS), wie bspw. OpenOlat oder Moodle integriert und veröffentlicht werden. H5P wurde entwickelt, um es Nutzern zu ermöglichen, ohne technisches Know-how interaktive Inhalte zu erstellen und zu teilen.

H5P unterstützt eine breite Palette an Inhaltstypen, z.B.:

  • Interaktive Videos und Bücher
  • Interaktive Präsentationen
  • Quizze (Single-Choice, Multiple-Choice, Lückentexte, Drag&Drop,...)
  • Spiele
  • 360° Videos
  • Timelines
  • u.v.m

Eine sehr gute Übersicht über die verfügbaren Inhaltstypen, inkl. Screencasts und Beispielen finden Sie unter folgendem Link: https://ebildungslabor.github.io/H5P

Eine der großen Stärken von H5P ist, dass es sehr benutzerfreundlich ist. Nutzer, die keine Programmierkenntnisse haben, können trotzdem interaktive Inhalte erstellen, indem sie die Vorlagen und Werkzeuge nutzen, die von H5P zur Verfügung gestellt werden.

ie kann man HP5-Dateien erstellen?
Für die Erstellung von H5P-Inhalten gibt es verschiedene Autorentools. Es gibt die Möglichkeit eine der browserbasierten Anwendungen zu nutzen oder die Destop-Version der Software LUMI lokal zu installieren.

Als Online-Autorentools stehen zur Verfügung:

  • https://h5p.org/
  • https://app.lumi.education/

Ein Vorteil dieser Tools ist, dass sie im Browser funktionieren und somit von jedem Gerät aus zugänglich sind. Das bedeutet, dass Nutzer ihre Inhalte von überall aus bearbeiten und teilen können. Die Nachteile hier sind allerdings, dass die Anwendungen zum Teil sehr Langsam sind und dass es Upload Beschränkung gibt. Des Weiteren kann man hier lediglich .h5p-Dateien abspeichern, die nicht überall integrierbar sind.

Ich nutze am liebsten die Desktop-Anwendung von LUMI. Diese kann man auf der Webseite von LUMI herunterladen: https://app.lumi.education/. Die Vorteile hier sind: es gibt keine Upload-Beschränkungen und man kann die erstellten H5P-Inhalte direkt als HTML-Datei oder Scorm-Paket exportieren. Scorm-Dateien können in jedes LMS eingebunden werden. Wenn Sie H5P einfach mal ausprobieren wollen, ohne sich zu registrieren oder etwa zu installieren, können Sie das auf einstiegh5p.de machen: https://einstiegh5p.de/.

Was ist SCORM?
SCORM (Sharable Content Object Reference Model) ist ein Standard für das Erstellen, Verpacken und Veröffentlichen von digitalen Lerninhalten. SCORM wurde entwickelt, um sicherzustellen, dass Lerninhalte, die in verschiedenen Lernmanagement-Systemen (LMS) verwendet werden, kompatibel und interoperabel sind. Kurz gesagt: Als Scorm-Paket abgespeicherte Lerninhalte können problemlos in nahezu jedes Learning.-Management-System integriert werden, ohne dass sie speziell angepasst werden müssen.

FAZIT
H5P eignet sich besonders gut für Anwender ohne Programmierkenntnisse, die interaktive Lerninhalte erstellen wollen. Lehrer und Bildungseinrichtungen können H5P kostenlos nutzen, um ihren Schülern und Studierenden interaktive Aufgaben und Lernspiele zur Verfügung zu stellen, die das Lernen unterstützen und gleichzeitig Spaß machen.
